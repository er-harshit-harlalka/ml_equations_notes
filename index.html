Regression

📈 Simple Linear Regression

Model Equation

Hypothesis
$$\hat{y} = \theta_0 + \theta_1 x$$
\(\theta_0\) = intercept, \(\theta_1\) = slope, \(x\) = feature

Parameter Estimates (Closed Form)

Slope
$$\theta_1 = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}$$
Intercept
$$\theta_0 = \bar{y} - \theta_1\bar{x}$$

Cost / Loss Function

Mean Squared Error Cost
$$J(\theta_0,\theta_1) = \frac{1}{2n}\sum_{i=1}^{n}\left(\hat{y}^{(i)} - y^{(i)}\right)^2$$

Gradient Descent Update

Update θ₀
$$\theta_0 \leftarrow \theta_0 - \alpha\frac{\partial J}{\partial \theta_0} = \theta_0 - \frac{\alpha}{n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})$$
Update θ₁
$$\theta_1 \leftarrow \theta_1 - \alpha\frac{\partial J}{\partial \theta_1} = \theta_1 - \frac{\alpha}{n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})x^{(i)}$$

📊 Multiple Linear Regression

Hypothesis (vector form)
$$\hat{y} = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_p x_p = \mathbf{X}\boldsymbol{\theta}$$
\(\mathbf{X} \in \mathbb{R}^{n \times (p+1)}\) includes bias column of 1s, \(\boldsymbol{\theta} \in \mathbb{R}^{p+1}\)

Cost Function

MSE Cost
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\|\mathbf{X}\boldsymbol{\theta} - \mathbf{y}\|^2$$

Gradient

Gradient w.r.t. θ
$$\nabla_{\boldsymbol{\theta}} J = \frac{1}{n}\mathbf{X}^\top(\mathbf{X}\boldsymbol{\theta} - \mathbf{y})$$

〰️ Polynomial Linear Regression

Hypothesis (degree d)
$$\hat{y} = \theta_0 + \theta_1 x + \theta_2 x^2 + \cdots + \theta_d x^d = \sum_{j=0}^{d}\theta_j x^j$$
Polynomial regression is still linear in parameters \(\boldsymbol{\theta}\). The same OLS / gradient descent applies after feature mapping \(\phi(x) = [1, x, x^2, \ldots, x^d]\).
Feature Map
$$\boldsymbol{\phi}(x) = \begin{bmatrix}1 \\ x \\ x^2 \\ \vdots \\ x^d\end{bmatrix}, \quad \hat{y} = \boldsymbol{\phi}(x)^\top\boldsymbol{\theta}$$

Cost Function

MSE Cost
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\sum_{i=1}^n\left(\sum_{j=0}^d \theta_j x_i^j - y_i\right)^2$$

🔢 Ordinary Least Squares (OLS)

Normal Equation (Closed Form Solution)
$$\hat{\boldsymbol{\theta}} = (\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top\mathbf{y}$$
Exact solution — no iterations. Requires \(\mathbf{X}^\top\mathbf{X}\) to be invertible. \(O(n p^2 + p^3)\) complexity.

Gauss-Markov Assumptions

Error / Residuals
$$\mathbf{e} = \mathbf{y} - \hat{\mathbf{y}} = \mathbf{y} - \mathbf{X}\hat{\boldsymbol{\theta}}$$ $$\mathbb{E}[\boldsymbol{\varepsilon}] = \mathbf{0}, \quad \text{Var}(\boldsymbol{\varepsilon}) = \sigma^2 \mathbf{I}$$
Hat Matrix (Projection)
$$\mathbf{H} = \mathbf{X}(\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top, \quad \hat{\mathbf{y}} = \mathbf{H}\mathbf{y}$$

🏔️ Ridge Regression (L2)

Cost / Loss Function
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\|\mathbf{X}\boldsymbol{\theta} - \mathbf{y}\|^2 + \lambda\|\boldsymbol{\theta}\|^2 = \frac{1}{2n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})^2 + \lambda\sum_{j=1}^p\theta_j^2$$
\(\lambda \geq 0\) is the regularisation strength. Bias term \(\theta_0\) is usually not penalised.
Closed Form Solution
$$\hat{\boldsymbol{\theta}}_{\text{ridge}} = (\mathbf{X}^\top\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^\top\mathbf{y}$$
Gradient
$$\nabla_{\boldsymbol{\theta}}J = \frac{1}{n}\mathbf{X}^\top(\mathbf{X}\boldsymbol{\theta}-\mathbf{y}) + 2\lambda\boldsymbol{\theta}$$
Ridge shrinks coefficients toward zero but never sets them exactly to zero. Useful when many features contribute weakly (multicollinearity).

🔪 Lasso Regression (L1)

Cost / Loss Function
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})^2 + \lambda\sum_{j=1}^p|\theta_j|$$
Subgradient
$$\frac{\partial J}{\partial \theta_j} = \frac{1}{n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})x_j^{(i)} + \lambda\,\text{sign}(\theta_j)$$ where \(\text{sign}(\theta_j) = \begin{cases}+1 & \theta_j > 0\\ [-1,1] & \theta_j=0\\ -1 & \theta_j<0\end{cases}\)
Soft-Thresholding Solution
$$\hat{\theta}_j = \text{sign}(\tilde{\theta}_j)\max\!\left(|\tilde{\theta}_j|-\lambda,\;0\right)$$
\(\tilde{\theta}_j\) = OLS estimate for feature \(j\)
Lasso promotes sparsity — drives some coefficients exactly to zero, performing automatic feature selection.

🌐 ElasticNet Regression

Cost / Loss Function
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\sum_{i=1}^n(\hat{y}^{(i)}-y^{(i)})^2 + \lambda_1\sum_{j=1}^p|\theta_j| + \lambda_2\sum_{j=1}^p\theta_j^2$$
Mixed Penalty (sklearn convention)
$$J(\boldsymbol{\theta}) = \frac{1}{2n}\|\mathbf{X}\boldsymbol{\theta}-\mathbf{y}\|^2 + \alpha\rho\|\boldsymbol{\theta}\|_1 + \frac{\alpha(1-\rho)}{2}\|\boldsymbol{\theta}\|_2^2$$
\(\alpha\) = total regularisation, \(\rho \in [0,1]\) = L1 ratio. \(\rho=1\) → Lasso, \(\rho=0\) → Ridge.
ElasticNet combines the sparsity of Lasso with the grouping effect of Ridge. Preferred when \(p \gg n\) or features are correlated.
Classification

📉 Logistic Regression

Sigmoid / Logistic Function
$$\sigma(z) = \frac{1}{1+e^{-z}}, \quad z = \boldsymbol{\theta}^\top\mathbf{x}$$
Hypothesis
$$\hat{y} = P(y=1|\mathbf{x};\boldsymbol{\theta}) = \sigma(\boldsymbol{\theta}^\top\mathbf{x})$$
Log-Odds (Logit)
$$\log\frac{P(y=1)}{P(y=0)} = \boldsymbol{\theta}^\top\mathbf{x}$$

Loss Function

Binary Cross-Entropy Loss
$$\mathcal{L}(\boldsymbol{\theta}) = -\frac{1}{n}\sum_{i=1}^n\left[y^{(i)}\log\hat{y}^{(i)} + (1-y^{(i)})\log(1-\hat{y}^{(i)})\right]$$

Gradient

Gradient
$$\nabla_{\boldsymbol{\theta}}\mathcal{L} = \frac{1}{n}\mathbf{X}^\top(\hat{\mathbf{y}}-\mathbf{y})$$

Multiclass (Softmax)

Softmax Output
$$P(y=k|\mathbf{x}) = \frac{e^{\boldsymbol{\theta}_k^\top\mathbf{x}}}{\sum_{j=1}^K e^{\boldsymbol{\theta}_j^\top\mathbf{x}}}$$

✂️ Support Vector Machine

SVM Classifier

Decision Boundary
$$\mathbf{w}^\top\mathbf{x} + b = 0$$
Margin
$$\text{Margin} = \frac{2}{\|\mathbf{w}\|}$$
Hard Margin Optimisation (Primal)
$$\min_{\mathbf{w},b}\;\frac{1}{2}\|\mathbf{w}\|^2 \quad\text{s.t.}\quad y^{(i)}(\mathbf{w}^\top\mathbf{x}^{(i)}+b)\geq 1\;\forall i$$
Soft Margin (Hinge Loss + Regularisation)
$$\min_{\mathbf{w},b,\boldsymbol{\xi}}\;\frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^n\xi_i$$ $$\text{s.t.}\quad y^{(i)}(\mathbf{w}^\top\mathbf{x}^{(i)}+b)\geq 1-\xi_i,\;\xi_i\geq 0$$
\(C\) controls regularisation strength. Larger \(C\) = less margin violation tolerance.
Hinge Loss
$$\mathcal{L}_{\text{hinge}} = \frac{1}{n}\sum_{i=1}^n\max\!\left(0,\,1-y^{(i)}(\mathbf{w}^\top\mathbf{x}^{(i)}+b)\right) + \frac{\lambda}{2}\|\mathbf{w}\|^2$$
Dual Problem
$$\max_{\boldsymbol{\alpha}}\sum_{i=1}^n\alpha_i - \frac{1}{2}\sum_{i=1}^n\sum_{j=1}^n\alpha_i\alpha_j y^{(i)}y^{(j)}\mathbf{x}^{(i)\top}\mathbf{x}^{(j)}$$ $$\text{s.t.}\;\sum_{i=1}^n\alpha_i y^{(i)}=0,\;0\leq\alpha_i\leq C$$
Prediction
$$\hat{y} = \text{sign}\!\left(\sum_{i\in\text{SV}}\alpha_i y^{(i)} K(\mathbf{x}^{(i)},\mathbf{x})+b\right)$$

SVR (Support Vector Regressor)

ε-Insensitive Loss
$$\mathcal{L}_\varepsilon(y,\hat{y}) = \max(0, |y-\hat{y}|-\varepsilon)$$
SVR Primal
$$\min_{\mathbf{w},b,\xi,\xi^*}\;\frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^n(\xi_i+\xi_i^*)$$ $$\text{s.t.}\;y_i-(\mathbf{w}^\top\mathbf{x}_i+b)\leq\varepsilon+\xi_i,\;(\mathbf{w}^\top\mathbf{x}_i+b)-y_i\leq\varepsilon+\xi_i^*,\;\xi_i,\xi_i^*\geq 0$$

🎲 Naïve Bayes

Bayes Theorem
$$P(y|\mathbf{x}) = \frac{P(\mathbf{x}|y)\,P(y)}{P(\mathbf{x})}$$
Naïve Independence Assumption
$$P(\mathbf{x}|y) = \prod_{j=1}^p P(x_j|y)$$
MAP Prediction
$$\hat{y} = \arg\max_y P(y)\prod_{j=1}^p P(x_j|y)$$

Bernoulli Naïve Bayes

Likelihood (binary features)
$$P(x_j|y) = p_{jy}^{x_j}(1-p_{jy})^{1-x_j}, \quad x_j\in\{0,1\}$$ $$p_{jy} = \frac{\text{count}(x_j=1 \;\text{and}\; y) + \alpha}{\text{count}(y) + 2\alpha} \quad\text{(Laplace smoothing)}$$

Multinomial Naïve Bayes

Likelihood (count features)
$$P(\mathbf{x}|y) = \frac{(\sum_j x_j)!}{\prod_j x_j!}\prod_j\theta_{jy}^{x_j}$$ $$\theta_{jy} = \frac{\sum_i x_{ij}\mathbf{1}[y_i=y] + \alpha}{\sum_k\sum_i x_{ik}\mathbf{1}[y_i=y] + p\alpha}$$

Gaussian Naïve Bayes

Likelihood (continuous features)
$$P(x_j|y) = \frac{1}{\sqrt{2\pi\sigma_{jy}^2}}\exp\!\left(-\frac{(x_j-\mu_{jy})^2}{2\sigma_{jy}^2}\right)$$
\(\mu_{jy},\sigma_{jy}^2\) estimated from training samples of class \(y\)

🔵 K-Nearest Neighbours

Euclidean Distance
$$d(\mathbf{x},\mathbf{x}') = \sqrt{\sum_{j=1}^p(x_j-x_j')^2} = \|\mathbf{x}-\mathbf{x}'\|_2$$
Minkowski Distance (generalised)
$$d_m(\mathbf{x},\mathbf{x}') = \left(\sum_{j=1}^p|x_j-x_j'|^m\right)^{1/m}$$
\(m=1\): Manhattan, \(m=2\): Euclidean, \(m\to\infty\): Chebyshev
KNN Classifier
$$\hat{y} = \underset{c}{\arg\max}\;\sum_{\mathbf{x}^{(i)}\in\mathcal{N}_k(\mathbf{x})}\mathbf{1}[y^{(i)}=c]$$
Majority vote over \(k\) nearest neighbours \(\mathcal{N}_k(\mathbf{x})\)
KNN Regressor
$$\hat{y} = \frac{1}{k}\sum_{\mathbf{x}^{(i)}\in\mathcal{N}_k(\mathbf{x})}y^{(i)}$$

KD-Tree

Space Partition
Binary tree splitting at each level on alternating dimensions. Split axis: \(j^* = (d \mod p)\). Split value: median of \(x_{j^*}\). Search: O(\(\log n\)) average, O(\(kn\)) worst. Efficient for \(p \ll 20\).

Ball-Tree

Ball Node
$$\text{Ball}(\mathbf{c},r) = \{\mathbf{x}: \|\mathbf{x}-\mathbf{c}\|\leq r\}$$
Centroid \(\mathbf{c}\), radius \(r = \max_i d(\mathbf{x}^{(i)},\mathbf{c})\). Better than KD-tree for high dimensions.
Tree Methods

🌳 Decision Trees

Impurity Measures

Gini Impurity
$$G = 1 - \sum_{k=1}^K p_k^2$$
Entropy
$$H = -\sum_{k=1}^K p_k\log_2 p_k$$
Information Gain
$$IG = H(\text{parent}) - \sum_{v}\frac{n_v}{n}H(\text{child}_v)$$
Split chosen to maximise \(IG\). \(p_k\) = fraction of samples of class \(k\) in node.
Gain Ratio (C4.5)
$$\text{GainRatio} = \frac{IG}{\text{SplitInfo}}, \quad \text{SplitInfo} = -\sum_v\frac{n_v}{n}\log_2\frac{n_v}{n}$$

Regression Tree

Variance Reduction / MSE Split Criterion
$$J = \frac{1}{n}\sum_{i=1}^n(y_i - \bar{y})^2 - \sum_v\frac{n_v}{n}\cdot\frac{1}{n_v}\sum_{i\in v}(y_i-\bar{y}_v)^2$$
Leaf Prediction
$$\hat{y}_{\text{leaf}} = \frac{1}{|R_m|}\sum_{i\in R_m}y_i$$

🌲🌲🌲 Random Forest

Bootstrap Aggregation (Bagging)
$$\hat{y}_{\text{RF}} = \frac{1}{B}\sum_{b=1}^B T_b(\mathbf{x})$$
Each tree \(T_b\) trained on bootstrap sample \(\mathcal{D}_b\) with random feature subset \(m \approx \sqrt{p}\) (classifier) or \(m \approx p/3\) (regressor).
Majority Vote (Classifier)
$$\hat{y} = \underset{c}{\arg\max}\;\sum_{b=1}^B\mathbf{1}[T_b(\mathbf{x})=c]$$
Out-of-Bag Error
$$\text{OOB Error} = \frac{1}{n}\sum_{i=1}^n\mathbf{1}\!\left[\hat{y}_i^{\text{OOB}}\neq y_i\right]$$
Estimate on samples not in bootstrap — free internal validation.
Feature Importance (MDI)
$$\text{FI}(j) = \frac{1}{B}\sum_{b=1}^B\sum_{t:\text{split on }j}\frac{n_t}{n}\cdot\Delta\text{impurity}(t)$$

⚡ AdaBoost

Algorithm
Initialise: \(w_i^{(1)} = 1/n\) for all \(i\)
For \(m=1\ldots M\):
$$\varepsilon_m = \sum_{i=1}^n w_i^{(m)}\mathbf{1}[h_m(\mathbf{x}^{(i)})\neq y^{(i)}]$$ $$\alpha_m = \frac{1}{2}\ln\!\left(\frac{1-\varepsilon_m}{\varepsilon_m}\right)$$ $$w_i^{(m+1)} = \frac{w_i^{(m)}\exp\!\left(-\alpha_m y^{(i)} h_m(\mathbf{x}^{(i)})\right)}{Z_m}$$
\(Z_m\) = normalisation constant. \(y\in\{-1,+1\}\).
Final Prediction
$$H(\mathbf{x}) = \text{sign}\!\left(\sum_{m=1}^M\alpha_m h_m(\mathbf{x})\right)$$
Exponential Loss
$$\mathcal{L}_{\text{exp}} = \sum_{i=1}^n\exp\!\left(-y^{(i)}\sum_m\alpha_m h_m(\mathbf{x}^{(i)})\right)$$

📈⚡ Gradient Boosting

Additive Model
$$F_M(\mathbf{x}) = F_0(\mathbf{x}) + \sum_{m=1}^M\eta\, T_m(\mathbf{x})$$
Pseudo-Residuals (Negative Gradient)
$$r_i^{(m)} = -\left[\frac{\partial \mathcal{L}(y_i, F(\mathbf{x}_i))}{\partial F(\mathbf{x}_i)}\right]_{F=F_{m-1}}$$
Fit weak learner \(T_m\) to these residuals at each stage.
MSE Loss Residuals
$$r_i = y_i - F_{m-1}(\mathbf{x}_i)$$
Line Search Step
$$\eta_m = \arg\min_\eta\sum_{i=1}^n\mathcal{L}(y_i,\; F_{m-1}(\mathbf{x}_i)+\eta T_m(\mathbf{x}_i))$$
Regularised Update
$$F_m(\mathbf{x}) = F_{m-1}(\mathbf{x}) + \eta\cdot T_m(\mathbf{x})$$
\(\eta\) = learning rate (shrinkage). Smaller \(\eta\) → better generalisation, more trees needed.

🚀 XGBoost

Regularised Objective
$$\mathcal{L}^{(m)} = \sum_{i=1}^n\left[\mathcal{L}(y_i, \hat{y}_i^{(m-1)}+f_m(\mathbf{x}_i))\right] + \Omega(f_m)$$ $$\Omega(f) = \gamma T + \frac{1}{2}\lambda\|\mathbf{w}\|^2$$
\(T\) = number of leaves, \(\mathbf{w}\) = leaf weights, \(\gamma\) = leaf complexity penalty
Second-Order Taylor Approximation
$$\mathcal{L}^{(m)} \approx \sum_{i=1}^n\left[g_i f_m(\mathbf{x}_i) + \frac{1}{2}h_i f_m^2(\mathbf{x}_i)\right] + \Omega(f_m)$$ $$g_i = \frac{\partial\mathcal{L}(y_i,\hat{y}^{(m-1)})}{\partial\hat{y}^{(m-1)}}, \quad h_i = \frac{\partial^2\mathcal{L}(y_i,\hat{y}^{(m-1)})}{\partial(\hat{y}^{(m-1)})^2}$$
Optimal Leaf Weight
$$w_j^* = -\frac{\sum_{i\in R_j}g_i}{\sum_{i\in R_j}h_i + \lambda}$$
Optimal Objective Value
$$\mathcal{L}^{*(m)} = -\frac{1}{2}\sum_{j=1}^T\frac{\left(\sum_{i\in R_j}g_i\right)^2}{\sum_{i\in R_j}h_i+\lambda} + \gamma T$$
Split Gain
$$\text{Gain} = \frac{1}{2}\left[\frac{G_L^2}{H_L+\lambda}+\frac{G_R^2}{H_R+\lambda}-\frac{(G_L+G_R)^2}{H_L+H_R+\lambda}\right]-\gamma$$
Clustering & Anomaly Detection

⭕ K-Means Clustering

Objective (WCSS / Inertia)
$$J = \sum_{k=1}^K\sum_{\mathbf{x}_i\in C_k}\|\mathbf{x}_i - \boldsymbol{\mu}_k\|^2$$
Centroid Update
$$\boldsymbol{\mu}_k = \frac{1}{|C_k|}\sum_{\mathbf{x}_i\in C_k}\mathbf{x}_i$$
Assignment Step
$$z_i = \underset{k}{\arg\min}\;\|\mathbf{x}_i - \boldsymbol{\mu}_k\|^2$$
Silhouette Score
$$s_i = \frac{b_i - a_i}{\max(a_i,\, b_i)}$$
\(a_i\) = mean intra-cluster distance, \(b_i\) = mean nearest-cluster distance. \(s\in[-1,1]\); higher → better.

Hierarchical (Agglomerative / Divisive)

Single Linkage
$$d(A,B) = \min_{\mathbf{a}\in A,\mathbf{b}\in B}\|\mathbf{a}-\mathbf{b}\|$$
Complete Linkage
$$d(A,B) = \max_{\mathbf{a}\in A,\mathbf{b}\in B}\|\mathbf{a}-\mathbf{b}\|$$
Average Linkage (UPGMA)
$$d(A,B) = \frac{1}{|A||B|}\sum_{\mathbf{a}\in A}\sum_{\mathbf{b}\in B}\|\mathbf{a}-\mathbf{b}\|$$
Ward Linkage
$$d(A,B) = \sqrt{\frac{2|A||B|}{|A|+|B|}}\|\boldsymbol{\mu}_A-\boldsymbol{\mu}_B\|$$

🌲🔴 Isolation Forest

Anomaly Score
$$s(\mathbf{x},n) = 2^{-\dfrac{\mathbb{E}[h(\mathbf{x})]}{c(n)}}$$
\(h(\mathbf{x})\) = path length from root to leaf for point \(\mathbf{x}\). \(c(n)\) = expected path length in unsuccessful BST search.
Expected Path Length Normaliser
$$c(n) = 2H(n-1) - \frac{2(n-1)}{n}, \quad H(i)=\ln(i)+0.5772\ldots\text{(Euler constant)}$$
Score \(s\approx 1\) → anomaly; \(s\approx 0.5\) → normal. Anomalies have shorter average path lengths because they are isolated quickly.

🔍 Local Outlier Factor (LOF)

Reachability Distance
$$\text{reach-dist}_k(\mathbf{x},\mathbf{o}) = \max\!\left(d_k(\mathbf{o}),\; d(\mathbf{x},\mathbf{o})\right)$$
\(d_k(\mathbf{o})\) = distance to \(k\)-th nearest neighbour of \(\mathbf{o}\)
Local Reachability Density
$$\text{lrd}_k(\mathbf{x}) = \left(\frac{1}{k}\sum_{\mathbf{o}\in\mathcal{N}_k(\mathbf{x})}\text{reach-dist}_k(\mathbf{x},\mathbf{o})\right)^{-1}$$
LOF Score
$$\text{LOF}_k(\mathbf{x}) = \frac{\dfrac{1}{k}\displaystyle\sum_{\mathbf{o}\in\mathcal{N}_k(\mathbf{x})}\text{lrd}_k(\mathbf{o})}{\text{lrd}_k(\mathbf{x})}$$
LOF \(\gg 1\) → outlier (less dense than neighbours). LOF \(\approx 1\) → inlier.

🫧 DBSCAN

ε-Neighbourhood
$$N_\varepsilon(\mathbf{x}) = \{\mathbf{x}'\in\mathcal{D} : d(\mathbf{x},\mathbf{x}')\leq\varepsilon\}$$
Core Point Condition
$$|N_\varepsilon(\mathbf{x})| \geq \text{MinPts}$$
Point Classification
Core Point: \(|N_\varepsilon(\mathbf{x})|\geq\text{MinPts}\)
Border Point: Not core, but in \(N_\varepsilon\) of a core point
Noise Point: Neither core nor border
Density-Reachability
\(\mathbf{x}'\) is directly density-reachable from core \(\mathbf{x}\) iff \(\mathbf{x}'\in N_\varepsilon(\mathbf{x})\). Chain of direct reachability → density-connected cluster.
Silhouette Score (cluster quality)
$$S = \frac{1}{n}\sum_{i=1}^n s_i, \quad s_i = \frac{b_i-a_i}{\max(a_i,b_i)}$$
Dimensionality Reduction

🔻 Principal Component Analysis (PCA)

Covariance Matrix
$$\boldsymbol{\Sigma} = \frac{1}{n-1}\sum_{i=1}^n(\mathbf{x}^{(i)}-\bar{\mathbf{x}})(\mathbf{x}^{(i)}-\bar{\mathbf{x}})^\top = \frac{1}{n-1}\tilde{\mathbf{X}}^\top\tilde{\mathbf{X}}$$
Eigendecomposition
$$\boldsymbol{\Sigma}\mathbf{v}_j = \lambda_j\mathbf{v}_j, \quad j=1,\ldots,p$$
Eigenvectors \(\mathbf{v}_j\) = principal components. Eigenvalues \(\lambda_j\) = variance explained.
SVD Approach
$$\tilde{\mathbf{X}} = \mathbf{U}\boldsymbol{\Sigma}_s\mathbf{V}^\top$$
Principal components = columns of \(\mathbf{V}\). Singular values \(\sigma_j = \sqrt{(n-1)\lambda_j}\).
Projection to d Dimensions
$$\mathbf{Z} = \tilde{\mathbf{X}}\mathbf{W}_d, \quad \mathbf{W}_d = [\mathbf{v}_1,\ldots,\mathbf{v}_d]\in\mathbb{R}^{p\times d}$$
Variance Explained Ratio
$$\text{VER}_d = \frac{\sum_{j=1}^d\lambda_j}{\sum_{j=1}^p\lambda_j}$$
Reconstruction Error
$$\mathcal{L}_{\text{PCA}} = \|\tilde{\mathbf{X}} - \mathbf{Z}\mathbf{W}_d^\top\|_F^2 = \sum_{j=d+1}^p\lambda_j$$

Kernel PCA

Kernel Matrix
$$\mathbf{K}_{ij} = k(\mathbf{x}^{(i)},\mathbf{x}^{(j)}) = \phi(\mathbf{x}^{(i)})^\top\phi(\mathbf{x}^{(j)})$$
Centred Kernel
$$\tilde{\mathbf{K}} = \mathbf{K} - \mathbf{1}_n\mathbf{K} - \mathbf{K}\mathbf{1}_n + \mathbf{1}_n\mathbf{K}\mathbf{1}_n, \quad \mathbf{1}_n=\tfrac{1}{n}\mathbf{1}\mathbf{1}^\top$$

Sparse PCA

Objective
$$\min_{\mathbf{A},\mathbf{B}}\|\tilde{\mathbf{X}}-\tilde{\mathbf{X}}\mathbf{B}\mathbf{A}^\top\|_F^2 + \lambda\|\mathbf{B}\|_1 \quad\text{s.t.}\;\mathbf{A}^\top\mathbf{A}=\mathbf{I}$$

Incremental / Online PCA

Processes data in mini-batches. Uses rank-1 updates to the SVD rather than full eigen-decomposition. Memory: O(\(d \cdot p\)) instead of O(\(n \cdot p\)).
Neural Networks

🧠 Perceptron

Net Input
$$z = \sum_{j=1}^p w_j x_j + b = \mathbf{w}^\top\mathbf{x}+b$$
Activation (Heaviside)
$$\hat{y} = \begin{cases}1 & z\geq 0\\ 0 & z<0\end{cases}$$
Weight Update (Perceptron Learning Rule)
$$w_j \leftarrow w_j + \eta(y-\hat{y})x_j, \quad b\leftarrow b+\eta(y-\hat{y})$$
Perceptron Loss
$$\mathcal{L} = \sum_{i\in\mathcal{M}}\left(-y^{(i)}\cdot(\mathbf{w}^\top\mathbf{x}^{(i)})\right)$$
\(\mathcal{M}\) = set of misclassified samples. \(y\in\{-1,+1\}\)

⚡ Activation Functions

Sigmoid
$$\sigma(z) = \frac{1}{1+e^{-z}}\in(0,1)$$ $$\sigma'(z) = \sigma(z)(1-\sigma(z))$$
Tanh
$$\tanh(z) = \frac{e^z-e^{-z}}{e^z+e^{-z}}\in(-1,1)$$ $$\tanh'(z) = 1-\tanh^2(z)$$
ReLU
$$\text{ReLU}(z) = \max(0,z)$$ $$\text{ReLU}'(z) = \begin{cases}1 & z>0\\ 0 & z\leq 0\end{cases}$$
Leaky ReLU
$$\text{LReLU}(z) = \begin{cases}z & z>0\\ \alpha z & z\leq 0\end{cases}, \;\alpha\approx 0.01$$ $$\text{LReLU}'(z) = \begin{cases}1 & z>0\\ \alpha & z\leq 0\end{cases}$$
ELU (Exponential Linear Unit)
$$\text{ELU}(z) = \begin{cases}z & z>0\\ \alpha(e^z-1) & z\leq 0\end{cases}$$ $$\text{ELU}'(z) = \begin{cases}1 & z>0\\ \text{ELU}(z)+\alpha & z\leq 0\end{cases}$$
Softmax (output layer)
$$\text{softmax}(\mathbf{z})_k = \frac{e^{z_k}}{\sum_{j=1}^K e^{z_j}}$$ $$\frac{\partial\text{softmax}_k}{\partial z_j} = \text{softmax}_k(\delta_{kj}-\text{softmax}_j)$$

💥 Neural Network Loss Functions

Binary Cross-Entropy
$$\mathcal{L}_{\text{BCE}} = -\frac{1}{n}\sum_{i=1}^n\left[y^{(i)}\log\hat{y}^{(i)} + (1-y^{(i)})\log(1-\hat{y}^{(i)})\right]$$
Used with sigmoid output. \(y\in\{0,1\}\).
Categorical Cross-Entropy
$$\mathcal{L}_{\text{CCE}} = -\frac{1}{n}\sum_{i=1}^n\sum_{k=1}^K y_{ik}\log\hat{y}_{ik}$$
Used with softmax output + one-hot labels. \(K\) classes.
Sparse Categorical Cross-Entropy
$$\mathcal{L}_{\text{SCCE}} = -\frac{1}{n}\sum_{i=1}^n\log\hat{y}_{i,c_i}$$
Same as CCE but labels are integer class indices \(c_i\) rather than one-hot vectors.

⚙️ Optimisers

Gradient Descent Variants

Batch Gradient Descent
$$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta\nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta};\mathcal{D})$$
Stochastic Gradient Descent (SGD)
$$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta\nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta};\mathbf{x}^{(i)},y^{(i)})$$
One sample per update. Noisy but fast; escapes local minima.
Mini-Batch SGD
$$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta\frac{1}{B}\sum_{i\in\mathcal{B}}\nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta};\mathbf{x}^{(i)},y^{(i)})$$
Batch size \(B\) typically 32–512. Balance between batch GD and SGD.

Momentum Methods

SGD with Momentum
$$\mathbf{v}_t = \beta\mathbf{v}_{t-1} + (1-\beta)\nabla_{\boldsymbol{\theta}}\mathcal{L}_t$$ $$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta\mathbf{v}_t$$
\(\beta\approx 0.9\). Accumulates velocity in consistent gradient directions.

Adaptive Methods

AdaGrad
$$\mathbf{G}_t = \mathbf{G}_{t-1} + \mathbf{g}_t\odot\mathbf{g}_t$$ $$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \frac{\eta}{\sqrt{\mathbf{G}_t}+\varepsilon}\odot\mathbf{g}_t$$
Accumulated squared gradients. Adapts per parameter. Can vanish learning rate.
RMSProp
$$\mathbf{v}_t = \beta\mathbf{v}_{t-1} + (1-\beta)\mathbf{g}_t^2$$ $$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \frac{\eta}{\sqrt{\mathbf{v}_t}+\varepsilon}\mathbf{g}_t$$
Exponential moving average of squared gradients. Fixes AdaGrad's diminishing LR.
Adam (Adaptive Moment Estimation)
$$\mathbf{m}_t = \beta_1\mathbf{m}_{t-1}+(1-\beta_1)\mathbf{g}_t \quad\text{(1st moment)}$$ $$\mathbf{v}_t = \beta_2\mathbf{v}_{t-1}+(1-\beta_2)\mathbf{g}_t^2 \quad\text{(2nd moment)}$$ $$\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1-\beta_1^t}, \quad \hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1-\beta_2^t} \quad\text{(bias correction)}$$ $$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t}+\varepsilon}\hat{\mathbf{m}}_t$$
Defaults: \(\beta_1=0.9,\;\beta_2=0.999,\;\varepsilon=10^{-8}\)
AdaDelta
$$\mathbf{E}[\mathbf{g}^2]_t = \rho\mathbf{E}[\mathbf{g}^2]_{t-1}+(1-\rho)\mathbf{g}_t^2$$ $$\Delta\boldsymbol{\theta}_t = -\frac{\sqrt{\mathbf{E}[\Delta\boldsymbol{\theta}^2]_{t-1}+\varepsilon}}{\sqrt{\mathbf{E}[\mathbf{g}^2]_t+\varepsilon}}\mathbf{g}_t$$ $$\mathbf{E}[\Delta\boldsymbol{\theta}^2]_t = \rho\mathbf{E}[\Delta\boldsymbol{\theta}^2]_{t-1}+(1-\rho)\Delta\boldsymbol{\theta}_t^2$$
No manual LR required. Ratio of RMS of updates to RMS of gradients.
Sequence Models

🔄 Recurrent Neural Network (RNN)

Hidden State Update
$$\mathbf{h}_t = \tanh(\mathbf{W}_{hh}\mathbf{h}_{t-1} + \mathbf{W}_{xh}\mathbf{x}_t + \mathbf{b}_h)$$
Output
$$\mathbf{y}_t = \mathbf{W}_{hy}\mathbf{h}_t + \mathbf{b}_y$$
BPTT Loss
$$\mathcal{L} = \sum_{t=1}^T\mathcal{L}_t, \quad \frac{\partial\mathcal{L}}{\partial\mathbf{W}} = \sum_{t=1}^T\frac{\partial\mathcal{L}_t}{\partial\mathbf{W}}$$
Vanilla RNNs suffer from vanishing/exploding gradients for long sequences. Use LSTM or GRU instead.

🔁 Long Short-Term Memory (LSTM)

Gates and State Equations
$$\mathbf{f}_t = \sigma(\mathbf{W}_f[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_f) \quad\text{(Forget gate)}$$ $$\mathbf{i}_t = \sigma(\mathbf{W}_i[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_i) \quad\text{(Input gate)}$$ $$\tilde{\mathbf{c}}_t = \tanh(\mathbf{W}_c[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_c) \quad\text{(Candidate)}$$ $$\mathbf{o}_t = \sigma(\mathbf{W}_o[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_o) \quad\text{(Output gate)}$$ $$\mathbf{c}_t = \mathbf{f}_t\odot\mathbf{c}_{t-1} + \mathbf{i}_t\odot\tilde{\mathbf{c}}_t \quad\text{(Cell state)}$$ $$\mathbf{h}_t = \mathbf{o}_t\odot\tanh(\mathbf{c}_t) \quad\text{(Hidden state)}$$
\(\odot\) = element-wise (Hadamard) product. \([\mathbf{h}_{t-1};\mathbf{x}_t]\) = concatenation. Forget gate prevents vanishing gradients through the cell state highway.

⚡🔁 Gated Recurrent Unit (GRU)

GRU Equations
$$\mathbf{z}_t = \sigma(\mathbf{W}_z[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_z) \quad\text{(Update gate)}$$ $$\mathbf{r}_t = \sigma(\mathbf{W}_r[\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_r) \quad\text{(Reset gate)}$$ $$\tilde{\mathbf{h}}_t = \tanh(\mathbf{W}_h[\mathbf{r}_t\odot\mathbf{h}_{t-1};\mathbf{x}_t]+\mathbf{b}_h) \quad\text{(Candidate)}$$ $$\mathbf{h}_t = (1-\mathbf{z}_t)\odot\mathbf{h}_{t-1} + \mathbf{z}_t\odot\tilde{\mathbf{h}}_t \quad\text{(Hidden state)}$$
GRU has fewer parameters than LSTM (no cell state, no output gate). Often comparable performance. Preferred for smaller datasets.

↔️ Bidirectional RNN

Forward Pass
$$\overrightarrow{\mathbf{h}}_t = f(\mathbf{W}_{xh}^\rightarrow\mathbf{x}_t + \mathbf{W}_{hh}^\rightarrow\overrightarrow{\mathbf{h}}_{t-1}+\mathbf{b}^\rightarrow)$$
Backward Pass
$$\overleftarrow{\mathbf{h}}_t = f(\mathbf{W}_{xh}^\leftarrow\mathbf{x}_t + \mathbf{W}_{hh}^\leftarrow\overleftarrow{\mathbf{h}}_{t+1}+\mathbf{b}^\leftarrow)$$
Combined Output
$$\mathbf{y}_t = \mathbf{W}_y[\overrightarrow{\mathbf{h}}_t;\overleftarrow{\mathbf{h}}_t] + \mathbf{b}_y$$
Concatenation of forward and backward hidden states. Context from both past and future tokens.

🔆 Attention Mechanism & KQV

Query, Key, Value Projections
$$\mathbf{Q} = \mathbf{X}\mathbf{W}^Q,\quad \mathbf{K} = \mathbf{X}\mathbf{W}^K,\quad \mathbf{V} = \mathbf{X}\mathbf{W}^V$$
\(\mathbf{X}\in\mathbb{R}^{n\times d_{\text{model}}}\), \(\mathbf{W}^Q,\mathbf{W}^K\in\mathbb{R}^{d_{\text{model}}\times d_k}\), \(\mathbf{W}^V\in\mathbb{R}^{d_{\text{model}}\times d_v}\)
Scaled Dot-Product Attention
$$\text{Attention}(\mathbf{Q},\mathbf{K},\mathbf{V}) = \text{softmax}\!\left(\frac{\mathbf{Q}\mathbf{K}^\top}{\sqrt{d_k}}\right)\mathbf{V}$$
Scaling by \(\sqrt{d_k}\) prevents softmax saturation in high dimensions.
Attention Weights
$$\mathbf{A} = \text{softmax}\!\left(\frac{\mathbf{Q}\mathbf{K}^\top}{\sqrt{d_k}}\right)\in\mathbb{R}^{n\times n}$$ $$A_{ij} = \frac{\exp(q_i^\top k_j/\sqrt{d_k})}{\sum_l\exp(q_i^\top k_l/\sqrt{d_k})}$$
Multi-Head Attention
$$\text{head}_h = \text{Attention}(\mathbf{Q}\mathbf{W}_h^Q, \mathbf{K}\mathbf{W}_h^K, \mathbf{V}\mathbf{W}_h^V)$$ $$\text{MultiHead}(\mathbf{Q},\mathbf{K},\mathbf{V}) = \text{Concat}(\text{head}_1,\ldots,\text{head}_H)\mathbf{W}^O$$
\(H\) heads, each with \(d_k = d_{\text{model}}/H\). \(\mathbf{W}^O\in\mathbb{R}^{Hd_v\times d_{\text{model}}}\).

Feed-Forward Sub-Layer
$$\text{FFN}(\mathbf{x}) = \max(0,\;\mathbf{x}\mathbf{W}_1+\mathbf{b}_1)\mathbf{W}_2+\mathbf{b}_2$$
\(\mathbf{W}_1\in\mathbb{R}^{d_{\text{model}}\times d_{ff}}\), \(d_{ff}=4d_{\text{model}}\) typically
Layer Normalisation
$$\text{LayerNorm}(\mathbf{x}) = \frac{\mathbf{x}-\mu}{\sqrt{\sigma^2+\varepsilon}}\odot\boldsymbol{\gamma}+\boldsymbol{\beta}$$
Residual Connections

📍 Positional Encoding

Sinusoidal Positional Encoding (Vaswani et al.)
$$PE_{(pos,2i)} = \sin\!\left(\frac{pos}{10000^{2i/d_{\text{model}}}}\right)$$ $$PE_{(pos,2i+1)} = \cos\!\left(\frac{pos}{10000^{2i/d_{\text{model}}}}\right)$$
\(pos\) = position in sequence, \(i\) = dimension index, \(d_{\text{model}}\) = embedding dimension.

$$\mathbf{X}' = \mathbf{X}_{\text{embed}} + \mathbf{PE}$$
Sinusoidal encoding enables the model to extrapolate to unseen sequence lengths and encodes relative position via dot products.

Kernel Functions

🌊 Kernel Functions

Linear Kernel
$$K(\mathbf{x},\mathbf{x}') = \mathbf{x}^\top\mathbf{x}'$$
Polynomial Kernel
$$K(\mathbf{x},\mathbf{x}') = (\mathbf{x}^\top\mathbf{x}'+c)^d, \quad c\geq 0,d\in\mathbb{Z}^+$$
RBF / Gaussian Kernel
$$K(\mathbf{x},\mathbf{x}') = \exp\!\left(-\frac{\|\mathbf{x}-\mathbf{x}'\|^2}{2\sigma^2}\right) = \exp(-\gamma\|\mathbf{x}-\mathbf{x}'\|^2)$$
\(\gamma = 1/(2\sigma^2)\). Most widely used kernel.
Sigmoid Kernel
$$K(\mathbf{x},\mathbf{x}') = \tanh(\kappa\,\mathbf{x}^\top\mathbf{x}'+c)$$
Valid kernel only for specific \(\kappa,c\). Related to neural network activations.
Mercer's Condition (valid kernel)
A function \(K:\mathcal{X}\times\mathcal{X}\to\mathbb{R}\) is a valid kernel iff its Gram matrix \(\mathbf{K}_{ij}=K(\mathbf{x}^{(i)},\mathbf{x}^{(j)})\) is positive semi-definite for all datasets.
Performance Metrics

🎯 Classification Metrics

Confusion Matrix

Confusion Matrix Predicted
Positive Negative
Actual Positive TP FN
Negative FP TN
Accuracy
$$\text{Acc} = \frac{TP+TN}{TP+TN+FP+FN}$$
Precision
$$P = \frac{TP}{TP+FP}$$
Recall (Sensitivity)
$$R = \frac{TP}{TP+FN}$$
Specificity
$$\text{Spec} = \frac{TN}{TN+FP}$$
F1 Score
$$F_1 = 2\cdot\frac{P\cdot R}{P+R} = \frac{2TP}{2TP+FP+FN}$$
F-Beta Score
$$F_\beta = (1+\beta^2)\cdot\frac{P\cdot R}{\beta^2 P+R}$$
\(\beta>1\): weight recall; \(\beta<1\): weight precision
Matthews Correlation Coefficient
$$\text{MCC} = \frac{TP\cdot TN - FP\cdot FN}{\sqrt{(TP+FP)(TP+FN)(TN+FP)(TN+FN)}}$$
ROC-AUC
$$\text{AUC} = \int_0^1 \text{TPR}\,d(\text{FPR}) = P(\hat{s}_+ > \hat{s}_-)$$
TPR = TP/P (Recall), FPR = FP/N. AUC = 1 → perfect, 0.5 → random.

📏 Regression Metrics

MAE
$$\text{MAE} = \frac{1}{n}\sum_{i=1}^n|y_i-\hat{y}_i|$$
MSE
$$\text{MSE} = \frac{1}{n}\sum_{i=1}^n(y_i-\hat{y}_i)^2$$
RMSE
$$\text{RMSE} = \sqrt{\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y}_i)^2}$$
MAPE
$$\text{MAPE} = \frac{100}{n}\sum_{i=1}^n\left|\frac{y_i-\hat{y}_i}{y_i}\right|\%$$
R-Squared (Coefficient of Determination)
$$R^2 = 1 - \frac{SS_{\text{res}}}{SS_{\text{tot}}} = 1 - \frac{\sum(y_i-\hat{y}_i)^2}{\sum(y_i-\bar{y})^2}$$
\(R^2=1\): perfect fit; \(R^2=0\): same as mean predictor; \(R^2<0\): worse than mean.
Adjusted R-Squared
$$\bar{R}^2 = 1-(1-R^2)\frac{n-1}{n-p-1}$$
Penalises for adding unnecessary features. \(p\) = number of predictors, \(n\) = sample size.
Bias-Variance Decomposition
$$\mathbb{E}[(y-\hat{y})^2] = \underbrace{\text{Bias}^2[\hat{y}]}_{\left(\mathbb{E}[\hat{y}]-y\right)^2} + \underbrace{\text{Var}[\hat{y}]}_{\mathbb{E}[(\hat{y}-\mathbb{E}[\hat{y}])^2]} + \underbrace{\sigma^2}_{\text{irreducible noise}}$$

🌿 Tree Criteria: Entropy, Gini, Information Gain

Shannon Entropy
$$H(S) = -\sum_{k=1}^K p_k\log_2 p_k, \quad p_k = \frac{|S_k|}{|S|}$$
Measures impurity/disorder. \(H=0\) → pure node; \(H=\log_2 K\) → maximally impure.
Gini Impurity
$$G(S) = 1-\sum_{k=1}^K p_k^2 = \sum_{k\neq k'} p_k p_{k'}$$
Probability that a random element would be incorrectly classified. Range \([0, 1-1/K]\).
Information Gain
$$IG(S,A) = H(S) - \sum_{v\in\text{vals}(A)}\frac{|S_v|}{|S|}H(S_v)$$
Gain Ratio (C4.5)
$$\text{GR}(S,A) = \frac{IG(S,A)}{-\sum_v\frac{|S_v|}{|S|}\log_2\frac{|S_v|}{|S|}}$$
